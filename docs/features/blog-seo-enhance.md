# Blog SEO & UX Enhancement

## 背景
博客 sund.site 目前 75 篇文章，SEO 基础设施（sitemap/RSS/OG）都有但不够精细。需要 4 项改进提升社交分享展示效果、搜索引擎收录质量、用户停留时长。

## 改动清单

### 1. 每篇文章独立的 OG Image
**现状：** 所有文章共享同一个 `images/share.png`（文件甚至不存在于磁盘）。社交分享卡片无图片或图片通用。
**目标：** 每篇文章分享到社交平台时，显示该文章首图作为 OG Image。若文章无图片，使用默认 fallback 图片。

### 2. 首页展示最近文章列表
**现状：** 首页仅有个人简介 + 项目链接，无文章入口。新访客到达后无法快速看到博客内容。
**目标：** 首页展示最近 5 篇文章列表（标题 + 日期），链接到文章详情页。

### 3. JSON-LD 结构化数据
**现状：** 无 `Article` / `BlogPosting` Schema。
**目标：** 每篇文章渲染时输出 JSON-LD 结构化数据，帮助 Google 理解文章内容、发布时间、作者等信息，提升富媒体摘要展示概率。

### 4. 文章底部相关推荐
**现状：** 文章阅读完即结束，无引导下一步阅读的链接。
**目标：** 每篇文章底部展示最多 3 篇相关文章（按 tags 匹配），若无相关则隐藏该区域。

## 验收条件
1. 分享「Hermes Agent」文章到 Telegram/微信，卡片显示该文章首图而非统一图片
2. 首页展示最近 5 篇文章列表
3. 查看页面源代码，每篇文章有 `<script type="application/ld+json">` 包含 Article schema
4. 在 AI 标签文章底部看到其他 AI 标签文章推荐
